# Documentation & Onboarding Skill

Create comprehensive documentation for developers

- Project completion
- Onboarding new developers
- API documentation needed
- Contributing guidelines

- README writing
- API documentation (OpenAPI)
- Architecture diagrams
- Development guides
- Contributing guidelines
