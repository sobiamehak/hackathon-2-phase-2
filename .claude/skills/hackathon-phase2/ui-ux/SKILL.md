# UI/UX Enhancement Skill

## Purpose
Create beautiful, accessible, delightful user interfaces

## When to Use
- Need design system
- Accessibility required
- Animations needed
- Dark mode

## Capabilities
- Design tokens
- Component library
- Accessibility (WCAG 2.1 AA)
- Animations (Framer Motion)
- Dark mode
- Responsive design

## Components
```typescript
- Button (with variants)
- Input (with validation states)
- Dialog (accessible)
- Toast notifications
- Loading skeletons
```

## Usage
```
Use @ui-ux skill for production-grade UI
```

## Commands
- "Create design system"
- "Build component library"
- "Add animations"
- "Implement dark mode"

## Accessibility Checklist
✓ Keyboard navigation
✓ Screen reader support
✓ Color contrast 4.5:1
✓ Focus indicators
