# Architecture Architect Skill

## Purpose
Design complete system architecture for full-stack applications

## When to Use
- Starting a new project
- Need system design
- API specification required
- Database schema design

## Capabilities
- Monorepo structure design
- RESTful API design
- Database schema design
- Authentication flow design
- Security model design

## Tech Stack
- Frontend: Next.js 16+
- Backend: FastAPI
- Database: PostgreSQL
- Auth: JWT (Better Auth)

## Deliverables
1. specs/architecture.md - System architecture
2. specs/api/rest-endpoints.md - API specification
3. specs/database/schema.md - Database schema

## Usage
```
Use @architecture skill to design the complete system architecture for a multi-user Todo application
```

## Commands
- "Design system architecture"
- "Create API specification"
- "Design database schema"
- "Document authentication flow"

## Output Structure
```
specs/
├── architecture.md
├── api/
│   └── rest-endpoints.md
└── database/
    └── schema.md
```
